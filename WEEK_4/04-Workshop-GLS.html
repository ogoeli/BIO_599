Modeling Heteroscedasticity with GLS

Xanthe Walker

2025-08-03

Overview

In this workshop, we explore how to detect and model heteroscedasticitya violation of the assumption that residuals have constant variancein linear models. Well use the nlme package to fit Generalized Least Squares (GLS) models that account for changing variance across groups or with continuous predictors.

We use two case studies:

  • Mandible length among golden jackal populations (heterogeneous group variances)
  • Sockeye salmon case study (variance modeled as a function of a continuous predictor)

Variance Structures in GLS

GLS models allow us to weight observations differently depending on variance structure. This helps improve inference and model fit when the assumption of constant variance is violated.

Here are common variance structures in ecological modeling:

Function Use Case Syntax
varIdent() Different variance per group weights = varIdent(form = ~1 | group)
varPower() Variance increases or decreases with a continuous covariate weights = varPower(form = ~ x)
varExp() Exponential change in variance with a covariate weights = varExp(form = ~ x)
varConstPower() Constant + power (good for small x-values) weights = varConstPower(form = ~ x)
varFixed() Known variances for each observation weights = varFixed(~ offset)
varComb() Combine group + continuous variance models weights = varComb(varIdent(...), varExp(...))

Part 1: Differences in Mandible Length Among Populations

Here we test if there are differences in mandible lengths between populations of golden jackals.

To demonstrate heterogeneity of variance, we simulate jaw length data for five jackal populations. Each population has its own mean and residual variance, simulating realistic differences in size and measurement precision.

set.seed(1)
ngroups <- 5
nsample <- 10
pop.means <- c(140, 145, 155, 140, 130)
sigma <- rep(c(16, 14, 12, 1.5, 11), each = nsample)
n <- ngroups * nsample
eps <- rnorm(n, 0, sigma)
x <- gl(ngroups, nsample, n, labels = LETTERS[1:5])
means <- rep(pop.means, each = nsample)
y <- as.numeric(means + eps)
jackal <- data.frame(jaw = y, pop = x)

Visualize Group Variance

ggplot(jackal, aes(x = pop, y = jaw, fill = pop)) + geom_boxplot() + theme_classic()+scale_fill_viridis_d()

There are visible differences in variation among groups. For regression models with categorical predictors, heterogeneity often shows up as differing spread in boxplots.

Fit a GLS model

gls.jackal <- gls(jaw ~ pop, weights = varIdent(form = ~1 | pop), data = jackal)

Using GLS gives us more flexibility and may improve coefficient precision when heteroscedasticity is present.

Part 2: Sockeye Salmon Case Study

This dataset reflects sockeye salmon (Oncorhynchus nerka) returning to the Fraser River system. We are interested in modeling how well counts at Mission (MisEsc) predict harvest plus escapement (SpnEsc).

salmon <- read.csv("../../Data files/Sockeyedata.csv")
ggplot(data = salmon, aes(x = MisEsc, y = SpnEsc, color = Year, shape = as.factor(Run))) +
  geom_point() +theme_bw()+
  labs(x = "Fish counted at Mission", y = "Harvest + Spawning Escapement")

Fit a linear model

lmsockeye <- lm(SpnEsc ~ MisEsc, data = salmon)
ggResidpanel::resid_panel(lmsockeye, plots = c("resid", "qq", "hist"), nrow = 1)

This model obviously violates the assumption of heterogeneity. We see a fan shape indicating that the variance increases with the number of fish passing Mission. The Q-Q plot and histogram of residuals also suggests that there are several large residuals, more than you would expect if the residuals came from a Normal distribution with constant variance.

Fit models with different variance structures

lmsockeye <- gls(SpnEsc ~ MisEsc, data = salmon)
fixedvar <- gls(SpnEsc ~ MisEsc, weights = varFixed(~ MisEsc), data = salmon)
varpow <- gls(SpnEsc ~ MisEsc, weights = varPower(form = ~ MisEsc), data = salmon)
varexp <- gls(SpnEsc ~ MisEsc, weights = varExp(form = ~ MisEsc), data = salmon) 
varconstp <- gls(SpnEsc ~ MisEsc, weights = varConstPower(form = ~ MisEsc), data = salmon)

Compare models

anova(lmsockeye, fixedvar, varpow, varexp, varconstp)

Interpretation: These models allow us to determine whether modeling variance as a function of fish count improves the fit. This is especially important when variance increases with the mean.

The constant + power model often performs well when the predictor can be near zero, offering a hybrid that handles increasing variance and base-level noise.

Check final model

plot(varconstp, plots = c("resid", "qq", "hist"), nrow = 3)

This model assumes residual variance increases with the covariate MisEsc. const (119.49) is the baseline variance. power (1.10) shows the rate of increase here, residual variance increases super-linearly with MisEsc. This confirms heteroskedasticity: observations with higher MisEsc values have more variable residuals.

Intercept: When MisEsc = 0, the predicted SpnEsc is -6.12. This isnt meaningful biologically (youd never have 0 fish at Mission), but it anchors the model.

Slope for MisEsc (0.828): For each additional fish counted at Mission, we predict ~0.83 additional fish in harvest + spawning escapement.

p-value for MisEsc < 0.001: This predictor is highly significant.

p-value for Intercept = 0.48: Not significant, but this is rarely of interest in ecological regressions.

The residual SD is low good model fit.
